# Carlaw Warehouse

This factory warehouse served as the primary rehearsal and recording space for the band during the Heartbeat Hotel period.

**Details:**
*   **Address:** 388 Carlaw.
*   **Significance:** The space facilitated the intense, focused creative output that led to the EP.
*   **Context:** This followed the move from the Toronto Bungalow.
