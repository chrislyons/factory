Data Flow Local-first CRUD, cloud sync, and sharing flows SEQUENCE 
sequenceDiagram
    participant User
    participant View
    participant Hook
    participant Service
    participant LocalStorage
    participant API
    participant R2

    Note over User,R2: CREATE LIST FLOW

    User->>View: Click "Create List"
    View->>Service: CryptoService.generateUUID()
    Service-->>View: listId
    View->>Service: StorageService.saveList(list)
    Service->>LocalStorage: setItem(listmaker_list:uuid)
    LocalStorage-->>Service: success
    View->>User: Navigate to /local/:id

    Note over User,R2: EDIT LIST FLOW (Local-First)

    User->>View: Edit cell value
    View->>View: Update React state
    View->>Service: StorageService.saveList(list)
    Service->>LocalStorage: setItem

    alt Cloud Sync Enabled
        View->>Hook: useCloudSync detects change
        Hook->>Service: EncryptionService.encrypt()
        Service->>API: POST /api/lists/:id
        API->>R2: PUT encrypted blob
        R2-->>API: success
        API-->>Service: {version, timestamp}
    end

    Note over User,R2: SHARE LIST FLOW

    User->>View: Click "Share"
    View->>Service: EncryptionService.generateKey()
    View->>Service: EncryptionService.encrypt(list, key)
    View->>API: POST /api/lists/create
    API->>R2: PUT encrypted blob
    API-->>View: {listId}
    View->>View: Build URL: /list/:id#key=...
    View-->>User: Copy share URL

    Note over User,R2: VIEW SHARED LIST FLOW

    User->>View: Open share URL
    View->>View: Extract key from URL fragment
    View->>API: GET /api/lists/:id
    API->>R2: GET encrypted blob
    API-->>View: encrypted data
    View->>Service: EncryptionService.decrypt()
    View-->>User: Show list content

Authentication & Authorization JWT auth, password protection, encrypted sharing SEQUENCE 
sequenceDiagram
    participant User
    participant Browser
    participant AuthContext
    participant API
    participant D1

    Note over User,D1: USER LOGIN FLOW

    User->>Browser: Submit login form
    Browser->>API: POST /api/auth/login
    API->>D1: SELECT user WHERE email
    D1-->>API: user row
    API->>API: Verify password (bcrypt)

    alt Password Valid
        API->>API: Generate JWT token
        API-->>Browser: {token, user}
        Browser->>AuthContext: login(token, user)
        AuthContext->>Browser: localStorage.setItem(token)
        Browser-->>User: Redirect to home
    else Password Invalid
        API-->>Browser: 401 Invalid credentials
    end

    Note over User,D1: SESSION VALIDATION

    Browser->>AuthContext: Check for stored token
    AuthContext->>Browser: localStorage.getItem(token)
    alt Token Exists
        Browser->>API: GET /api/auth/me
        API->>API: Verify JWT signature
        alt Token Valid
            API->>D1: SELECT user WHERE id
            API-->>Browser: {user}
        else Token Expired
            API-->>Browser: 401 Unauthorized
            Browser->>AuthContext: logout()
        end
    end

    Note over User,D1: ENCRYPTED SHARING FLOW

    User->>Browser: Click "Share" on list
    Browser->>Browser: EncryptionService.generateKey()
    Note right of Browser: AES-256-GCM via Web Crypto
    Browser->>Browser: EncryptionService.encrypt(list)
    Browser->>API: POST /api/lists/create
    API-->>Browser: {listId}
    Browser->>Browser: Build URL with key in fragment
    Note right of Browser: /list/{id}#key={base64}
    Browser-->>User: Copy share URL
