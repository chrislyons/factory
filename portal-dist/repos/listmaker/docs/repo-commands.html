listmaker - Command Reference

React + TypeScript + Vite development commands

Last updated: 2025-11-02

ℹ️ Tech Stack
Package manager: npm | Framework: React 18 | Build tool: Vite | Testing: Vitest | Language: TypeScript | Deploy: Cloudflare Pages
⚠️ Local Use Only
This page is for local development reference. Click any command to copy it to clipboard.
Job Command Description
Setup & Installation
Install dependencies npm install Install all project dependencies from package.json
Add dependency npm install <package> Install new runtime dependency
Add dev dependency npm install --save-dev <package> Install new development dependency
Remove dependency npm uninstall <package> Uninstall dependency and update package.json
Update dependencies npm update Update all dependencies to latest compatible versions
Development
Start dev server npm run dev Start Vite development server with hot reload (default: http://localhost:5173)
Preview build npm run preview Preview production build locally at http://localhost:4173
Type check tsc --noEmit Run TypeScript compiler in check-only mode (no output)
Lint code npm run lint Run ESLint to check for code quality issues
Building
Build for production npm run build Run type check + build optimized production bundle to dist/
Build with analysis npm run build:analyze Build with bundle size analysis and visualizations
Clean build output rm -rf dist/ Remove build artifacts before fresh build
Analyze bundle size npm run build && du -sh dist/* Build and show size of output files
Rebuild and preview npm run build && npm run preview Full rebuild and start preview server
Testing
Run all tests npm test Run all tests with Vitest in watch mode
Test UI npm run test:ui Open Vitest UI for interactive test exploration
Test coverage npm run test:coverage Generate code coverage report
Run specific test npm test <filename> Run tests in specific file
Cloudflare Pages Deployment
Deploy to preview npm run deploy:preview Build and deploy to preview environment (testing)
Deploy to production npm run deploy:prod Build and deploy to production (main branch)
Quick deploy npm run deploy Build and deploy (defaults to preview)
Login to Cloudflare wrangler login Authenticate Wrangler CLI with Cloudflare account
List deployments wrangler pages deployment list --project-name=listmaker Show recent deployments with IDs
View deployment logs wrangler pages deployment tail --project-name=listmaker Stream real-time deployment logs
Rollback deployment wrangler pages deployment rollback <ID> Revert to previous deployment by ID
Create R2 buckets wrangler r2 bucket create listmaker-storage && wrangler r2 bucket create listmaker-storage-preview Create R2 buckets for cloud sync (one-time setup)
List R2 buckets wrangler r2 bucket list Show all R2 buckets in account
Set Pages secret wrangler pages secret put STRIPE_SECRET_KEY Store environment variable securely
Documentation
List documentation ls -1 docs/lst/ | grep -E '^LST[0-9]{3,4}\s+' | sort List all LST-prefixed documentation files
Find next doc number ls -1 docs/lst/ | grep -E '^LST[0-9]{3}' | sed -E 's/LST([0-9]+).*/\1/' | sort -n | tail -1 Get highest LST document number
View INDEX cat docs/lst/INDEX.md Display documentation index
Validation & Cleanup
Check config hierarchy ~/dev/scripts/validate-config-hierarchy.sh Validate Claude Code configuration inheritance
Check for clutter ~/dev/scripts/find-root-clutter.sh Find misplaced documentation files
Check PREFIX collisions ~/dev/scripts/check-prefix-collisions.sh --verbose Verify no PREFIX conflicts across repos
Archive old docs ~/dev/scripts/archive-old-docs.sh Move docs older than 180 days to archive/
Clean node_modules rm -rf node_modules/ && npm install Remove and reinstall all dependencies (fresh start)
Clean build cache rm -rf dist/ node_modules/.vite Remove build output and Vite cache
Debugging
Check package info npm list List all installed packages and versions
Check specific package npm list <package> Show version and dependencies of specific package
Check outdated packages npm outdated List packages with available updates
Verify Vite config cat vite.config.ts Display Vite configuration
Check TypeScript config cat tsconfig.json Display TypeScript configuration
Project Information
Show project structure tree -L 2 -I 'node_modules|dist' Display directory tree (excluding build artifacts)
Count TypeScript files find src -name '*.ts' -o -name '*.tsx' | wc -l Count number of TypeScript source files
Count test files find tests -name '*.test.ts' -o -name '*.test.tsx' | wc -l Count number of test files
Lines of code find src -name '*.ts' -o -name '*.tsx' | xargs wc -l Count total lines of TypeScript code
