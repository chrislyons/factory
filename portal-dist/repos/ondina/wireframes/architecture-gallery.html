iOS Sync Flow

Sequence

Offline-first bidirectional sync

sequenceDiagram participant UI as SwiftUI View participant VM as ViewModel participant Repo as Repository participant DB as GRDB/SQLite participant Sync as SyncEngine participant API as API Server Note over UI,API: Offline-First Write Flow rect rgb(40, 70, 50) UI->>VM: Submit reflection VM->>Repo: saveMorningReflection() Repo->>DB: INSERT (needs_sync=true) DB-->>Repo: Success Repo-->>VM: Saved locally VM-->>UI: Show success ✓ end rect rgb(60, 50, 40) Note right of Sync: Background sync Sync->>Repo: getPendingSyncReflections() Repo->>DB: SELECT WHERE needs_sync=true DB-->>Repo: [pending items] Sync->>API: POST /api/reflections alt Success API-->>Sync: 200 OK Sync->>Repo: markSynced() Repo->>DB: UPDATE needs_sync=false else Conflict (409) API-->>Sync: 409 Conflict Note right of Sync: Server wins - will merge on pull else Offline API-->>Sync: Network error Note right of Sync: Keep needs_sync=true end end Note over UI,API: Pull Flow rect rgb(40, 60, 80) Sync->>Repo: getCursor(reflections) Repo->>DB: SELECT cursor Sync->>API: GET /sync/reflections?cursor=X API-->>Sync: {items, newCursor} loop Each item Sync->>Repo: mergeFromRemote(item) Repo->>DB: UPSERT if version newer end Sync->>Repo: updateCursor(newCursor) end
