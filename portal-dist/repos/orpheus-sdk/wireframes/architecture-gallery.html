Lock-free callbacks, true-peak metering, shmui visualization

sequenceDiagram participant U as User participant UI as UI Thread participant Q as SPSC Queue participant T as Transport participant R as RoutingMatrix participant M as TruePeakMeter participant AA as shmui::AudioAnalyzer participant V as shmui Visualizers participant D as Audio Driver Note over U,D: Initialization U->>UI: Load Session UI->>T: new TransportController T->>R: new RoutingMatrix(config) R->>M: Init per-channel meters Note over U,D: Audio Processing (every 5ms) D->>T: processAudio(buffer) Note right of T: Read samples from clips Note right of T: Apply fades per clip T->>R: Add to input buffer Note right of R: GainSmoother processing R->>M: 4x oversample metering Note right of R: Group mixing + headroom Note right of R: Soft-knee limiter R-->>T: Output buffer alt Clip reached end alt Loop enabled T->>Q: postCallback(onLooped) else No loop T->>Q: postCallback(onFinished) end end Note over U,D: UI Callback (10Hz) UI->>Q: processCallbacks() Q-->>UI: callback UI->>U: Update UI state Note over U,D: shmui Visualization (60 FPS) D->>AA: pushSamples(buffer) Note right of AA: Lock-free ring buffer V->>AA: getFFTData() / getRMS() AA-->>V: FFTData / BandLevels V->>U: Paint visualizers
