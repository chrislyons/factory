3 Threading Model

▼
Three-thread architecture ensuring real-time audio performance. The Message Thread handles UI events, the Audio Thread processes audio with zero allocations (no locks, no I/O), and the File I/O Thread handles background loading. Communication between threads uses lock-free queues and atomic operations only.
