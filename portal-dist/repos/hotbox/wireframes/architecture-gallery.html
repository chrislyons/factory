Data Flow Chat request/response cycle SEQUENCE 
sequenceDiagram
    participant User
    participant UI as React UI

    participant V as Vectorize
    participant KV
    participant LLM as OpenAI

    User->>UI: Type query
    UI->>W: POST /api/chat/stream

    par Hybrid Retrieval
        W->>V: Dense search (top-12)
        V-->>W: Passages
    and
        W->>KV: BM25 sparse search
        KV-->>W: Scores
    end

    W->>W: hybridMerge + rerank (top-5)
    W->>LLM: Chat completion + context
    LLM-->>W: Answer + citations

    W->>W: calculateCoverage

    alt Coverage below 40%
        W->>LLM: Retry with gpt-4o
        LLM-->>W: Better answer
    end

    W->>W: validateCopyright
    W-->>UI: Stream JSON deltas
    UI-->>User: Display with [n] citations

Authentication & Authorization Token gating, access control SEQUENCE 
sequenceDiagram
    participant User
    participant UI as React UI
    participant W as Worker
    participant V as Vectorize

    Note over User,V: Public Reader (No Auth)
    User->>UI: Visit /
    UI->>W: POST /api/chat/stream
    W->>V: Query (Read permission)
    V-->>W: Results
    W-->>UI: Stream response

    Note over User,V: Debug Routes (Token-Gated)
    User->>UI: Visit /author
    UI->>W: GET /api/debug/vectorize

    alt DEBUG_MODE=local
        W-->>UI: Debug data (no token)
    else DEBUG_MODE=shared
        alt Valid token
            W-->>UI: Debug data
        else Invalid token
            W-->>UI: HTTP 403
        end
    end

    Note over User,V: Author Routes (Token-Gated)
    User->>UI: Click Compose
    UI->>W: POST /api/author/compose
Authorization: Bearer token

    alt Token valid
        W-->>UI: Pipeline response
    else Token invalid
        W-->>UI: HTTP 403
    end
