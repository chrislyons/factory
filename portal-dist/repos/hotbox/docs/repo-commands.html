Job Command Description
Setup & Installation
Install dependencies npm install Install all package dependencies
Beat Pack Development
Build beat packs npm run build:beatpacks Generate beat pack JSON bundles from beats.yaml and claims.csv
Compose drafts npm run compose LLM-generated drafts from beat packs (auto/manual mode)
Stitch chapter npm run stitch Merge beat drafts into cohesive chapter
Validate chapters npm run validate:chapters Check A-claim usage, citation coverage ≥85%
Content Ingestion & Retrieval
Ingest passages npm run ingest Embed passages for retrieval (Vectorize)
Ingest beat packs npm run ingest:beatpacks Embed beat packs for composition retrieval
Full ingest pipeline npm run ingest && npm run ingest:beatpacks Run both passage and beat pack ingestion
Development
Start dev server npm run dev Start Cloudflare Worker + web app (Wrangler dev)
Start HTTP pipeline npm run http Start local Ollama pipeline server (port 3001)
Test chat endpoint curl -X POST http://localhost:8787/api/chat/stream?container_id=TEST -d '{"query":"test"}' Test streaming chat endpoint
Debug coverage curl http://localhost:8787/api/debug/nodal-01 Debug endpoint (requires HOTBOX_DEBUG_MODE=local)
Testing & Quality
Test worker npm run test:worker Run Cloudflare Worker tests
Test web app npm run test:web Run web application tests
Eval search npm run eval:search Evaluate search quality and coverage
Type check npm run typecheck Run TypeScript type checking
Lint code npm run lint Run ESLint on codebase
Format code npm run fmt Format code with Prettier
Ollama (Free Local LLM)
Install Ollama curl -fsSL https://ollama.com/install.sh | sh Install Ollama on macOS/Linux
Pull model ollama pull llama3.2 Download Llama 3.2 model
Configure pipeline export LLM_PROVIDER=ollama && export OLLAMA_MODEL=llama3.2 Set environment variables for Ollama
Configure worker echo "AUTHOR_PIPELINE_BASE=http://localhost:3001" >> app/worker/.dev.vars Point worker to local Ollama pipeline
Git Operations
Check status git status View current branch and changed files
View diff git diff See unstaged changes
View recent commits git log --oneline -10 Show last 10 commits
Pull latest git pull origin main Fetch and merge latest changes
Cloudflare & Vectorize
Export API token export CLOUDFLARE_API_TOKEN="$(grep CLOUDFLARE_API_TOKEN app/worker/.dev.vars | cut -d= -f2)" Export token for wrangler CLI (required before wrangler commands)
Verify auth wrangler whoami Check Cloudflare authentication status
Upload vectors wrangler vectorize insert hotbox-index --file content/chapters/TNC001/.build/beatpack_vectors/vectors.jsonl Upload vectors to Cloudflare Vectorize index
Check env bindings npm run diag:env Verify Cloudflare KV, DO, R2, Vectorize bindings
Check worker health npm run diag:bindings Probe /api/health endpoint (requires npm run dev)
Embeddings Providers
Use local embeddings EMBEDDINGS_PROVIDER=local npm run ingest Deterministic hash-based embeddings (256 dims, no API cost, for CI/testing)
Use OpenAI embeddings EMBEDDINGS_PROVIDER=openai npm run ingest Semantic embeddings via OpenAI (1536 dims, requires OPENAI_API_KEY)
Use Ollama embeddings EMBEDDINGS_PROVIDER=ollama npm run ingest Free semantic embeddings via local Ollama (nomic-embed-text default, OLMo 3.1 planned)
Ollama with custom model EMBEDDINGS_PROVIDER=ollama OLLAMA_EMBED_MODEL=mxbai-embed-large npm run ingest Use specific Ollama embedding model
Full local pipeline EMBEDDINGS_PROVIDER=local npm run ingest && EMBEDDINGS_PROVIDER=local npm run ingest:beatpacks Ingest passages + beat packs with local embeddings
Diagnostics
Check Node version node --version Verify Node.js version (require 20.x)
Check npm version npm --version Verify npm version
List workspaces npm run workspaces list Show all workspace packages
