nmos-vault-sync Commands

Installation

Install dependencies
cd nmos-vault-sync && npm install
Build TypeScript
npm run build

Sync Operations

One-time sync SAFE
npm run sync
Dry run (preview changes) SAFE
npm run sync -- --dry-run
Watch mode (continuous sync) MODIFIES VAULT
npm run dev -- watch
Custom config file
npm run sync -- --config ./config/custom.yaml

Discovery & Debugging

Discover NMOS/VSM endpoints SAFE
npm run sync -- discover
Dump NMOS state to JSON SAFE
npm run sync -- dump --output data/samples/state.json
Run tests
npm test

Python Scripts (Legacy)

Rebuild all streams MODIFIES VAULT
python3 scripts/rebuild_streams_complete.py

python3 scripts/create_agw_entities.py
Create card children MODIFIES VAULT
python3 scripts/create_card_children.py
Add router targets MODIFIES VAULT
python3 scripts/add_router_targets.py

Vault Operations

File Statistics

Count all markdown files SAFE
find . -name "*.md" -type f | wc -l
Count streams by router SAFE
for r in AMX101 AMX102 AMX103 AMX104; do echo -n "$r: "; ls streams/Stream-$r-*.md 2>/dev/null | wc -l; done
List gateways SAFE
ls -1 gateways/

Search Operations

Find streams by gateway SAFE
grep -l "TORMPXAGW005" streams/*.md
Find ACR40 entities SAFE
grep -l "acr: ACR40" streams/*.md router-cards/*.md

Git Operations

Status check SAFE
git status
Commit vault changes LOCAL
git add . && git commit -m "chore: Sync vault from NMOS"
Create save point SAFE
git checkout -b save/description-$(date +%Y%m%d-%H%M)
