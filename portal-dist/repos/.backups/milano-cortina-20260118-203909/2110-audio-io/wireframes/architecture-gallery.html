    Note over Node,Receiver: IS-04: Discovery & Registration

    Node->>Node: DNS-SD Query
_nmos-register._tcp
    Node->>Registry: POST /x-nmos/registration/v1.3
Register Node, Device, Senders, Receivers
    Registry-->>Node: 201 Created

    Note over Node,Receiver: IS-05: Connection Management

    Controller->>Registry: GET /x-nmos/query/v1.3/senders
    Registry-->>Controller: List of available senders

    Controller->>Node: PATCH /x-nmos/connection/v1.1/single/receivers/{id}/staged
    Note right of Controller: Set transport_file (SDP)
Set master_enable: true
    Node-->>Controller: 200 OK

    Controller->>Node: POST /x-nmos/connection/v1.1/single/receivers/{id}/active
    Node->>Receiver: Subscribe to ST2110 multicast
    Receiver-->>Node: Audio stream active

    Note over Node,Receiver: IS-08: Audio Channel Mapping

    Controller->>Node: PATCH /x-nmos/channelmapping/v1.0/map/activations
    Note right of Controller: Remap channels:
Input 1-8 → Output 1-8
    Node-->>Controller: 200 OK
    Node->>Node: Apply channel routing

I/O Card Types Audio Gateway card variants and protocols CLASS 
classDiagram
    direction LR

    class IOCard {
        <>
        +String slot: A-Z
        +Number inputs
        +Number outputs
        +Array streams
    }

    class AES {
        Type: AES3/EBU Digital
        Channels: Variable
        Format: 24-bit @ 48kHz
        Connector: XLR or BNC
        Cards: 40 AES, 42 AES
    }

    class Dante {
        Type: Audinate Dante
        Channels: 64 in/out
        Format: 24-bit @ 48kHz
        Transport: Layer 3 IP
        Cards: 40 Dante 1, 40 Dante 2
    }

    class SDI {
        Type: Embedded Audio
        Channels: 64 in/out
        Format: 16 channels per SDI
        Standard: SMPTE 299M
        Cards: 40 SDI, 42 SDI
    }

    class MADI {
        Type: AES10 MADI
        Channels: 64 in/out
        Format: 24-bit @ 48kHz
        Transport: Coax or Fiber
        Cards: 40 ProTools, 40 UTL MADI 01-06
    }

    class MicLine {
        Type: Analog I/O
        Channels: 48 in/16 out
        Format: +4dBu/-10dBV
        Preamps: Remote controlled
        Cards: Stagebox cards
    }

    class Waves {
        Type: Waves SoundGrid
        Channels: 64 in/out
        Format: 24-bit @ 48/96kHz
        Transport: Ethernet
        Cards: 40 Waves, 42 Waves
    }

    class Studer {
        Type: Studer D21m
        Channels: 64 in/out
        Format: Proprietary
        Transport: MADI bridge
        Cards: 40 StuderA, 40 StuderB
    }

        +String type: router-card

    class Stream {
        +String type: stream
        +String router: AMX###
        +Number physical_output
        +String acr: ACR40|ACR42

        8 channels per stream
    }

    class AudioGateway {
        +String type: agw
        +String hostname
        +Array~Slot~ slots
        +Array~Port~ ports
        Examples: TORMPXAGW005-012
    }

    class Box {
        +String type: box
        +String acr: ACR40|ACR42
        +Number boxId
        +String model
        +String location
        +NetworkInfo network
        Examples: 40 AUDIO RACK
    }

    class Card {
        +String type: card
        +String parent: gateway
        +String cardType
        +Number inputs
        +Number outputs
        +String slot
        +Array~Stream~ streams
    }

    class Slot {
        +String label: A-Z
        +Card card
        Physical position in AGW
    }

    class Port {
        +Number portNum: 1-64
        +Stream send
        +Stream receive
    }

    class Route {
        +String direction: IN|OUT
        +Device device
        +Number port
        +String slot
        +String cardType
        +String channels
    }

    RouterCard "1" --> "*" Stream : produces
    Stream "*" --> "1" AudioGateway : routes to
    AudioGateway "1" --> "*" Slot : contains
    AudioGateway "1" --> "*" Port : has
    Slot "1" --> "1" Card : holds
    Port "1" --> "2" Stream : send/receive
    Card "1" --> "*" Stream : streams
    AudioGateway "*" --> "*" Box : manages
    Stream "1" --> "1" Route : defines
Data Flow Audio signal paths (send/receive) SEQUENCE 
sequenceDiagram
    participant Console as Calrec Apollo Console
    participant Core as Impulse Core Router
    participant Router as Router Card (AMX###)
    participant Stream as ST2110 Stream
    participant AGW as Audio Gateway
    participant Card as I/O Card
    participant IO as Physical I/O

    Note over Console,IO: SEND PATH (Console to External)

    Console->>Core: Audio bus output
    Core->>Router: Route decision
    Router->>Stream: Assign to stream (8ch)
    Stream->>AGW: ST2110 transport
    AGW->>Card: Port mapping
    Card->>IO: D/A conversion
    IO-->>IO: To external device

    Note over Console,IO: RECEIVE PATH (External to Console)

    IO->>Card: Analog/digital input
    Card->>AGW: A/D conversion
    AGW->>Stream: ST2110 transport
    Stream->>Router: Stream assignment
    Router->>Core: Route decision
    Core->>Console: Audio bus input
