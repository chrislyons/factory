OSC Control — Tier 1 Parameters
Set density oscsend localhost 9000 /vmotes/density f 0.8 Particle concentration (0.01-1.0)
Set scale oscsend localhost 9000 /vmotes/scale f 2.0 Vessel size (0.1-5.0)
Set luminosity oscsend localhost 9000 /vmotes/luminosity f 2.5 Brightness (0.0-3.0, no rebuild)
Set fill oscsend localhost 9000 /vmotes/fill f 0.6 Volumetric fill (0.0-1.0)
Set force oscsend localhost 9000 /vmotes/force f 0.5 Centripetal gravity (0.0-1.0)
Set seed oscsend localhost 9000 /vmotes/seed i 12345 PRNG seed (uint32)
Set theme oscsend localhost 9000 /vmotes/theme s hologram Theme (hologram/photorealistic/monochrome)
Set shape oscsend localhost 9000 /vmotes/shape s "Torus Knot" Vessel preset
Enable quantize H oscsend localhost 9000 /vmotes/quantizeH i 1 Horizontal lattice snap
OSC Control — Tier 2 Expressive
Scatter oscsend localhost 9000 /vmotes/scatter f 0.5 Per-vmote random displacement (0-1)
Spin oscsend localhost 9000 /vmotes/spin f 0.8 Cloud Y-axis rotation rad/s (-2 to 2)
Tilt oscsend localhost 9000 /vmotes/tilt f 0.3 Cloud X-axis tilt (-1 to 1)
Orbit oscsend localhost 9000 /vmotes/orbit f 0.4 Per-vmote orbital animation (0-1)
Pulse oscsend localhost 9000 /vmotes/pulse f 0.6 Rhythmic scale breathing (0-1)
Hue shift oscsend localhost 9000 /vmotes/hue f 0.3 HSL hue shift (-1 to 1)
Saturation oscsend localhost 9000 /vmotes/sat f 1.5 Saturation multiplier (0-2)
Emit oscsend localhost 9000 /vmotes/emit f 0.5 Particle visibility fraction (0-1)
Camera distance oscsend localhost 9000 /vmotes/cam_dist f 8.0 Camera distance (1-20)
Camera height oscsend localhost 9000 /vmotes/cam_height f 3.0 Camera Y position (-5 to 10)
Camera spin oscsend localhost 9000 /vmotes/cam_spin f 0.2 Camera auto-orbit (-2 to 2)
OSC Control — FFT & Envelopes
FFT kick band oscsend localhost 9000 /vmotes/fft/kick f 0.9 Kick energy (0-1, maps to density+scale)
FFT hi band oscsend localhost 9000 /vmotes/fft/hi f 0.7 Hi-hat energy (0-1, maps to scatter)
FFT mid band oscsend localhost 9000 /vmotes/fft/mid f 0.5 Mid energy (0-1, maps to luminosity)
FFT air band oscsend localhost 9000 /vmotes/fft/air f 0.3 Air energy (0-1, maps to orbit)
FFT sub band oscsend localhost 9000 /vmotes/fft/sub f 0.8 Sub-bass energy (0-1, maps to scale+force)
Envelope trigger oscsend localhost 9000 /vmotes/envelope/trigger s density f 0.9 f 0.05 f 0.2 f 0.6 f 0.5 ADSR envelope (param, target, A, D, S, R)
Envelope release oscsend localhost 9000 /vmotes/envelope/release s density Release envelope for param
Strudel Integration
Strudel pattern (direct) oscsend localhost 9000 /dirt/play s density f 0.8 s lum f 2.0 s spin f 0.5 Strudel /dirt/play with visual params
Connect Strudel headless strudel_play_headless('s("bd").density(0.8).lum(2)', 120, '127.0.0.1', 9000) Start Strudel headless scheduler targeting vmotes relay
Stop Strudel headless strudel_stop_headless() Stop headless scheduler
