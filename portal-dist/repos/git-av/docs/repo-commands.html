Job Command Description
Setup
Install (standard) pip install -e . Install git-av in editable/development mode
Install (dev) pip install -e ".[dev]" Install with dev dependencies (pytest, ruff, coverage)
Install (C2PA) pip install -e ".[c2pa]" Install with C2PA provenance dependencies
Install (OTIO) pip install -e ".[otio]" Install with OpenTimelineIO support (Phase 5)
Verify install gav --version Check installed version (should show 0.3.0-alpha)
Install hooks cp .git-hooks-template/pre-commit .git/hooks/ && cp .git-hooks-template/post-commit .git/hooks/ && chmod +x .git/hooks/{pre,post}-commit Install C2PA git hooks (auto-manifest + metadata injection)
Usage (gav CLI)
Init gav init Initialize a new gav repository in current directory
Snapshot gav snapshot file.wav -m "Description" Create a versioned snapshot of a file
Status gav status Show working tree status (tracked files, changes)
Revert (list) gav revert file.wav --list List available versions for a file
Revert gav revert file.wav 2 Revert file to version 2 (instant via symlink)
Branch create gav branch feature Create a new branch named "feature"
Checkout gav checkout feature Switch branch via symlink swizzling (<1s)
C2PA generate gav c2pa generate <blob_hash> Generate C2PA provenance manifest for a blob
C2PA verify gav c2pa verify <blob_hash> Verify C2PA manifest integrity
C2PA inspect gav c2pa inspect <blob_hash> Show detailed provenance information
C2PA audit gav c2pa audit Full vault compliance report
C2PA status gav c2pa status Show C2PA tooling status
Help gav --help Show all available commands and options
Testing
All tests pytest tests/ -v Run full test suite (47 tests) with verbose output
Unit tests pytest tests/unit/ -v Run unit tests only (fast, isolated)
Integration tests pytest tests/integration/ -v Run integration tests (end-to-end workflows)
Coverage pytest tests/ --cov=src/gav --cov-report=html Run tests with HTML coverage report
Single file pytest tests/unit/test_symlink_manager.py -v Run a single test file (replace filename as needed)
Lint ruff check src/ Run ruff linter on source code (line-length: 100)
Debugging
Direct invoke python -m gav.cli.main --help Run CLI directly via Python module (for debugging)
PDB debug python -m pdb -m gav.cli.main snapshot file.wav -m "Test" Run CLI with Python debugger attached
Inspect DB sqlite3 .gav_ledger.db Open ledger database for manual inspection
List tables sqlite3 .gav_ledger.db ".tables" Show all tables in the ledger database
Show branches sqlite3 .gav_ledger.db "SELECT * FROM branches;" Query all branches from ledger
Git
Status git status Show working tree status

Diff staged git diff --staged Show staged changes
Log git log --oneline -20 Show last 20 commits (compact)
Fetch git fetch --all Fetch all remotes
Pull git pull --rebase Pull with rebase to keep history linear
